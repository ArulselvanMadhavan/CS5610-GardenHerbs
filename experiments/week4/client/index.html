Purpose of the Experiment: To design the front end of a single page application using MEAN Stack.

What does experiment intend to achieve? This experiment is intended to achieve the front end design needed for the application.

1. First , Mark the HTML container that needs to controlled by Angular controller and declare the controller. 

        < div ng-controller="course-rate-ctrl">  --> HTML element that will be controlled by a Angular Controller.

2. Populate the list of courses available in NEU , from the pre-defiened controller. In this experiment, we will use a pre-defined array but in the next experiment , we will retrieve this content from the server.

Shown below is the code that iterates over the 'courses' array and updates the HTML element.
        < tr ng-repeat="course in courses">
        < td>
        < div class="pull-left">
        < strong>{{course.number}}< /strong>< br />
        < small>{{course.title}}< /small>
                            < /div>

3. Next we provide the users the option of adding courses to the list. To achieve this, we declare an object called newcourse and assign them with 2 attributes - title and name , which in turn is taken as input from the user.

            < form role="form" data-ng-submit="addcourse(newcourse.number, newcourse.title)">
            < div class="row">
            < div class="col-sm-5">
            < label class="sr-only" for="number">Number< /label>
            < input type="text" class="form-control" name="number" placeholder="Enter the Course number that you wanna add"
                   data-ng-model="newcourse.number" autofocus />
                < /div>
            < div class="col-sm-5">
            < input type="text" class="form-control" name="title" placeholder="Enter the name of the course that you wanna add"
                   data-ng-model="newcourse.title" />
                < /div>
            < div class="col-sm-2">
            < button type="submit" class="btn btn-default form-control">
                        Add
                    < /button>
                < /div>
            < /div>
            < div class="alert alert-info" ng-if="!courses.length">
            < strong>First! No courses are available for rating at this moment.
            < /div>
        < /form>

4. When the user clicks the add button the form is submitted and the values in the textboxes are fed into the controller , which then maps these values to the course list array.

Note: This addition of courses to the array is not permanent. To save the courses permanently , we would need connection to the mongo DB. We will set up the Mongo DB connection to this application in the subsequent experiments.

5. Next, we provide users with the option to delete courses from the course list.

            < div class="pull-right">
            < button type="button" class="btn btn-danger pull-right" ng-click="deletecourse($index)">
                                    Delete
            < /button>
            < /div>

6. When the user clicks the delete button , 'deletecourse' function is called with the index value of the course on which the delete button was clicked.

7. The controller takes care of deleting the course from the list displayed to the user.

Note: This deletion of courses to the array is not permanent. To save the changes permanently , we would need connection to the mongo DB. We will set up the Mongo DB connection to this application in the subsequent experiments.

8. The controller is alerted when all the courses are deleted and there are no courses to display. Controller handles this situation by rendering a new view that represents the empty course list.

Next Phase of this application : We will configure and deploy this file in the server built using node js.
