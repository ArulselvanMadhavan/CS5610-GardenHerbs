Documentation:

Purpose of the Experiment: To get hands on experience in using Angular JS to manipulate HTML elements. This experiments acts an Angular JS refresher, so as to implement the MEAN Stack , in the later experiments.

What does this experiment intend to achieve? In this experiment, I have tried to use Angular JS directives to manipulate the HTML elements by retrieving data from them and updating data to them.

1. First , declare a container as Angular app , so that the browser understands that the elements that are under this tag has to be treated seperately.

2. Mark , the elements that you wish to manipulate as ng-model.

3. The value of ng-model elements can be accessed using {{name of the data model}}

4. Perform calulation on these data models and display them to the user.

        Here is the sample code that perform these operations.
             < input type="text" data-ng-model="kilometers" />< /p>       
            < p>The temperature in celsius would be {{(kilometers) * 0.62}}< /p>
            < h2>Miles to Kilometers< /h2>
            < p>Input a temperature in Miles: < input type="text" data-ng-model="miles" />< /p>
            < p>The temperature in celsius would be {{miles * 1.61}}< /p>
