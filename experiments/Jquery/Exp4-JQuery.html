Experiment4:Using JQuery to dynamically create,modify and delete a HTML element

Experiment Demo

Click anywhere in the grey highlighted area

Documentation:

Purpose of the Experiment:To learn to use the JQuery libraries to model the HTML elements and the CSS of HTML elements.

What does this experiment intend to achieve? When the user clicks in the grey highlighted area, a circle expands and fades dynamically.
This experiment dynamically creates a HTML element based on the position of the user clicks, apply style to it and remove that element after a specific time period.

Steps:

1.Using JQuery, Set the width and height of the 'touch area' when the document load completes.

2.When a click event happens within the 'touch-area', create a new

element that will hold the circle style effect that is about to be generated using a CSS style class - '.circle'

3.Here is the code of the CSS class '.circle' and the Jquery that creates a div element for a circle and maps it to the appropriate stylesheets.
Inline comments are added to the code to make the code-self-explanatory. 

 $(document).ready(function () {
 $('.touch-area').width($(window).width());
 $('.touch-area').height(60);
 var index = 0;
 $('.touch-area').on('click', function (e) {
 var circle_ident = 'circle_' + index;
 var circle_ident_div = div class="circle ' + circle_ident + '"> div
 var dot_circle_ident = '.' + circle_ident;
 $('.touch-area').append(circle_ident_div); /* Append the circle div to the touch area div*/
 $(dot_circle_ident).offset({ top: e.pageY, left: e.pageX }); /*Get the co-ordinates of the click event and set it as the current co-ordinates of the circle*/
 $(dot_circle_ident).addClass('expandandfade'); /* Add the expand and fade class to the new created circle div element*/
 setTimeout(function () {
 $(dot_circle_ident).remove();}, 1000);/* remove the created circle element after 1 sec */
                index++;
            });
        });
